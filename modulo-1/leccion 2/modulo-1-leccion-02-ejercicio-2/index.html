Web Dev: Adalab Style

Adalab gives women the opportunity to retrain themselves as frontend developers in just 3 months. In order to do this, they must master several programming languages. To start, students learn to master HTML in order to build the basic structure and content for their websites.

Once they've got a handle on HTML, studentes will tackle CSS. Cascading Style Sheets work in conjunction with HTML to take websites from merely functional to absolutely fabulous!

The third programming language that Adalab students must learn is JavaScript. This programming language takes things to another level, allowing for websites to be more attractive, as well as easier to use, with more responsive, interactive elements. Students will also practice using the JavaScript library React.

Other programming languages used in web development:

  • Java
  • C++
  • Ruby
  • Python
